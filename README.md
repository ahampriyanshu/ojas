## Installation

```bash

git clone https://github.com/ahampriyanshu/ojas.git
pip install -r requirements.txt
python3 manage.py migrate
python3 manage.py runserver

```

## Deployment

* Set ``DEBUG = False`` 
* Change the ``SECRET_KEY``
* Run ``python3 manage.py collectstatic``
